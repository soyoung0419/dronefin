| **요구사항 고유번호** | INR-021 |
| **요구사항 명칭**     | 시각적으로 차별화된 학과 정체성 표현 |

| **요구사항 세부내용** | - 학과만의 상징 컬러, 패턴 또는 캐릭터 디자인 활용<br> - 학과 비전이나 슬로건이 반영된 이미지/도식을 삽입<br> - 졸업생 인터뷰, 수상 이력, 연구성과 강조 섹션 제작<br> - 특성화 분야(예: AI, IoT 등)를 시각화한 배너나 섹션 구성 |
